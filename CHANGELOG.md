# Changelog

All notable changes to the Kanban Board project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-17

### Added
- ✅ Core kanban board functionality with drag and drop
- ✅ Task creation, editing, and deletion
- ✅ Priority system (High, Medium, Low)
- ✅ Due date management and overdue tracking
- ✅ Search and filter functionality
- ✅ Multiple view filters (All, Today, Overdue, High Priority)
- ✅ Undo/Redo functionality with full history
- ✅ Auto-save to localStorage
- ✅ Export board as JSON
- ✅ Import board from JSON
- ✅ Dark and Light theme support
- ✅ Real-time board statistics
- ✅ Toast notifications for user feedback
- ✅ Keyboard shortcuts support
- ✅ SVG icons throughout the UI
- ✅ Responsive design for all screen sizes
- ✅ Sidebar navigation
- ✅ Filter tabs for quick view switching
- ✅ Modern animations and transitions
- ✅ Help and keyboard shortcuts guide

### Features
- Drag and drop tasks between columns
- Real-time search with instant filtering
- Priority color indicators
- Overdue task highlighting
- Board statistics dashboard
- Undo/Redo with Ctrl+Z/Ctrl+Y shortcuts
- Theme switching with persistent preferences
- Auto-save to localStorage without manual action
- Export/Import for data backup and sharing
- Toast notifications for all user actions
- Responsive layout for mobile and desktop

### Documentation
- Created comprehensive README.md
- Added project structure guidelines
- Documented all features and usage
- Included keyboard shortcuts reference
- Added customization guide
- Troubleshooting section
